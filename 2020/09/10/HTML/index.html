html文档是由标签构成，每个标签都有开始标签和结束标签，开始标签可以声明属性，结束标签以/结尾
html ：
doctype：声明当前文档使用的html版本，避免出现兼容性问题
结构性标签；
html：
1.1 head：html的文档根标签，一个文档只能有一个根标签
1.2 head：是html的自标签，用来说明文档元信息
1.2.1 meta：用来设置文档的字符集，保证文档不会乱码，还可以用来设置viewport，一般在开发移动端网页的时候需要。
1.2.2 title：用来设置页面的标题明，title和meta标签还可以用来设置seo关键字
1.2.3 link： 用来导入外部的css文件
1.2.4 style： style标签内部用来定义内部样式。
1.2.5 script：用来导入外部的js文件，也可以在script的内部编写js文件

#body：
1.布局标签：1.1 div标签用来定义文档中的分区或节，可以将文档分为独立的、不同的部分
1.2 span标签被用来组合文档中的行内元素
2.显示数据的标签：
2.1标题标签 h1-h6
2.2文字段落标签：p
2.3显示图片标签：img（str）
2.4超链接：a
2.4.1 herf属性指定要跳转的目标文件路径
2.4.2 内容是用来给用户点击的
2.5表格 table
2.5.1 thead表格头部 tr th
2.5.2 tbody表格内容 tr td
2.6列表
2.6.1 有序列表 ol li
2.6.2 无序列表 ul li
2.6.3 自定义列表dl dt、dd
3.接受输入的标签
3.1 form：表单容器
3.2 表单元素
3.2.1 input type属性有23个值，如text单行文本框、password密码框、radio单选按钮、checkbox复选按钮、file选择文件
3.2.2 select元素用来给用户提供一个下拉列表 option标签用来定义列表中的可用选项
3.3 textarea：定义多行的文本输入控件
3.4 表单元素要注意的地方
3.4.1 一般表单元素可以使用label来辅助用户点击
3.4.2 单行文本框和密码框都可以有placeholder属性设置输入提示信息
3.4.3 一般表单元素应该有name元素，提交指定的参数的名称
3.4.5表单元素的value属性用来设置初始值，获取用户输入值
3.4.6设置初始值
3.4.6.1在input中如果为radio，checkbox使用chexked属性，否则就直接设置value属性
3.4.6.2在select中在option元素中添加selected
3.4.6.3在texarea中在元素内容中设置初始值
3.4.7所有的标签都应该有开始和结束标签，如果这个标签没有内容，可以在开始标签的结束位置添加/；textarea，script标签中必须有结束标签
