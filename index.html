CSS

css是cascading style sheet；html元素的外观是由样式属性值决定的，可以通过内联样式、内部样式、外部样式三种不同的方式来设置元素的样式属性值，如果一个元素同一个样式属性在多个不同的地方设置了值，最具体的位置样式属性值生效。
css定义位置：1、元素的style属性
2、文档的style标签里
3、外部文档
#选择器：选择器是用来描述要应用样式的元素，每个html元素都有一个标签名，还可以有class属性，id属性，css选择器里有元素选择器、id选择器、class选择器

  1. 如果一个选择器的名字是一个html标签名，那么这是一个元素选择器，他定义的样式会应用到和他名称相同的html元素上
  2. id选择器以#开头，作用于id属性和他名称相同的元素
  3. class选择器以.开头，作用于class属性包含他的名称的元素

盒模型

1.内容：width,height
2.内填充：padding,padding-top,padding-right,padding-bottom,padding-left
3.边框：border,border-top,border-right,border-bottom,border-left,border-style,border-width,border-color
4.外填充：margin,margin-top,margin-right,margin-bottom,margin-left
5.标准盒模型
6.怪异盒模型（ie）

    2.4. background-orient

使用”伪类“超链接的样式

超链接有四种状态：正常、鼠标悬浮、鼠标按下、访问过
可以通过伪类来给这四种状态分别设置样式
l:link:连接
v:visited,访问过
h:hover,鼠标悬浮
a:active,鼠标按下

元素的显示方式

在css中，万物皆块，display的值决定了这个块有不同的特性,同一个元素可以通过设置display的值来在去不同的显示方式中切换
display:

  1. none,不显示
  2. block,块级元素，单独占有一行，可以设置宽高,h1-h6,p,div,ul,ol,li,dl,dt,dd等的display值都是block
  3. inline,行内元素，可以和其他行内元素共处一行，不能设置宽高,span,a,img,i,input,select,textarea等都是inline元素
  4. inline-block,行内块元素，可以和其他行内块元素及行内元素共处一行，可以设置宽高
  5. flex,弹性盒子

浮动

1.元素可以浮动
2.如果一个元素浮动后，那么对他后面的元素有影响，对前面的元素没有影响
3.一个块级元素的宽默认是父元素的宽，高默认是由子元素撑起来的。一个行内元素的宽和高是由他的内容决定。
4.一个块级元素的子元素要是浮动了，那么他就不会撑起父元素的高了。

定位

在html中，元素的关系是父子关系和兄弟关系。浏览器是按照元素出现的嵌套关系和先后的关系来渲染页面元素。这样的一个渲染的过程就是文档流。
一个元素在页面上的位置和他与其他元素的关系有关。有的时候需要将一个元素的位置进行特殊的定位，需要使用position属性。
position的值

  1. static:是默认值，在文档流中就是这个值。
  2. relative：相对自身原来的位置定位,这个元素没有脱离文档路I，对文档流中其他的元素没有影响
  3. fixed：相对浏览器窗口来定位,脱离了文档流
  4. absolute：相对第一个已经定位（position属性值不是static）的祖先元素定位，脱离文档流
    4.1. 如果没有祖先元素定位，就会相对浏览器窗口定位
    4.2. 如果有祖先元素使用了定位，就会相对第一个（从内到外）使用了定位的祖先元素定位

在已经使用了position属性后，使用top,right,bottom,left来完成定位。

HTML

html文档是由标签构成，每个标签都有开始标签和结束标签，开始标签可以声明属性，结束标签以/结尾
html ：
doctype：声明当前文档使用的html版本，避免出现兼容性问题
结构性标签；
html：
1.1 head：html的文档根标签，一个文档只能有一个根标签
1.2 head：是html的自标签，用来说明文档元信息
1.2.1 meta：用来设置文档的字符集，保证文档不会乱码，还可以用来设置viewport，一般在开发移动端网页的时候需要。
1.2.2 title：用来设置页面的标题明，title和meta标签还可以用来设置seo关键字
1.2.3 link： 用来导入外部的css文件
1.2.4 style： style标签内部用来定义内部样式。
1.2.5 script：用来导入外部的js文件，也可以在script的内部编写js文件

#body：
1.布局标签：1.1 div标签用来定义文档中的分区或节，可以将文档分为独立的、不同的部分
1.2 span标签被用来组合文档中的行内元素
2.显示数据的标签：
2.1标题标签 h1-h6
2.2文字段落标签：p
2.3显示图片标签：img（str）
2.4超链接：a
2.4.1 herf属性指定要跳转的目标文件路径
2.4.2 内容是用来给用户点击的
2.5表格 table
2.5.1 thead表格头部 tr th
2.5.2 tbody表格内容 tr td
2.6列表
2.6.1 有序列表 ol li
2.6.2 无序列表 ul li
2.6.3 自定义列表dl dt、dd
3.接受输入的标签
3.1 form：表单容器
3.2 表单元素
3.2.1 input type属性有23个值，如text单行文本框、password密码框、radio单选按钮、checkbox复选按钮、file选择文件
3.2.2 select元素用来给用户提供一个下拉列表 option标签用来定义列表中的可用选项
3.3 textarea：定义多行的文本输入控件
3.4 表单元素要注意的地方
3.4.1 一般表单元素可以使用label来辅助用户点击
3.4.2 单行文本框和密码框都可以有placeholder属性设置输入提示信息
3.4.3 一般表单元素应该有name元素，提交指定的参数的名称
3.4.5表单元素的value属性用来设置初始值，获取用户输入值
3.4.6设置初始值
3.4.6.1在input中如果为radio，checkbox使用chexked属性，否则就直接设置value属性
3.4.6.2在select中在option元素中添加selected
3.4.6.3在texarea中在元素内容中设置初始值
3.4.7所有的标签都应该有开始和结束标签，如果这个标签没有内容，可以在开始标签的结束位置添加/；textarea，script标签中必须有结束标签
